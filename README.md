# Phone-Detection
## Why
At Westwood High School students are supposed to put their phones into the phone "cubbies." Despite this requirement, many students do not comply or forget, leading teachers to take valuable time that they could be teaching to cross refrence attendance and the phone "cubbies." As part of an AP CSA project we decided to see if we could mitigate this waste of time. 

## How
First we attached ArUco markers to the corners of the phone "cubby." Then using the OpenCV Library we were able to digitally flatten an image. This mitigrates the role that the angle of which the picture is taken plays in wraping the image. We then scaled the image to 1000x1500. 
For this project we were tasked with using Generative AI to assist us. We mainly used Gemini, as WHS uses Google Workspace.
